# Quocproject
Quocproject
StellarFund là một smart contract đơn giản trên mạng Stellar giúp bất kỳ ai cũng có thể dễ dàng nộp tiền XLM (native) vào quỹ chung. Mỗi lần nộp được cố định ở mức 5000 stroops (0.0005 XLM). Dự án nhằm minh họa và làm nền tảng cho các ứng dụng quyên góp, quỹ cộng đồng, hoặc tích lũy quỹ nhỏ trên Stellar.
